CURSO DE JAVASCRIPT: AULA #03 - OS TIPOS DE VALORES QUE TEMOS EM JAVASCRIPT!

Tipos de valores:

Primitive / Value Types
  • String
  • Number
  • Boolean
  • Undefined
  • Null

Exemplos:

let name = "Marcos"; //variável do tipo String

let idade = 39; //variável do tipo Number

let isApproved = true; // variável do tipo Boolean, true or false

let firstName; // variavél do tipo Undefined

let selectedIngredient = null; // variável do tipo Null
